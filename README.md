# STEAM-Summer-Camp

## GitHub Pages Deployment

1. Push code to GitHub:
   - `git add .`
   - `git commit -m "Initial STEAM site"
   - `git push origin main`
2. In repository Settings > Pages:
   - Source: `main` branch
   - Folder: `/ (root)`
   - Save
3. Visit: `https://<your-username>.github.io/STEAM-Summer-Camp`

## Project structure

- `index.html` — Main site with STEAM tab navigation
- `styles.css` — STEAM theme and tab style
- `script.js` — Tab switching + keyboard navigation
