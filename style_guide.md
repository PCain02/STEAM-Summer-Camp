# STEAM Camp Website — LLM Style Guide

## Design Intent
Create a modular, tab-based interface inspired by a file system or code editor, combined with playful building block visuals for a kid-friendly STEAM camp experience.

### Audience
- Primary: Kids (grades 4–6)
- Secondary: Parents and educators

### Tone
- Playful but organized  
- Creative but not chaotic  
- Educational but exciting  

### Constraints
- Avoid emojis entirely
- Maintain clean, professional readability
- Prioritize accessibility and clarity over decoration

---

## Core Visual System

### STEAM Color Mapping
Each tab and its content is themed consistently:

- S (Science) → #4CD964 (Green)
- T (Technology) → #9B6BFF (Purple)
- E (Engineering) → #FF9F43 (Orange)
- A (Art) → #FFD93D (Yellow)
- M (Math) → #4F9DFF (Blue)

### Neutral Colors
- Background → #F8FAFC
- Surface → #FFFFFF
- Section Alt → #EEF2F7
- Text Primary → #1F2937
- Text Secondary → #6B7280
- Border → #E5E7EB

---

## Navigation System

### Tab-Based Navigation (File System Inspired)

#### Behavior
- Top navigation uses horizontal tabs
- Each tab represents a STEAM category
- Active tab appears raised and connected to content
- Inactive tabs appear behind the active tab
